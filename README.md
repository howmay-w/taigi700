因為家己 beh 練習，所以整理各位先進 ê 資源、koh 請 Claude 寫 99% 程式碼所創出來 ê 家私，希望會對有仝款需求 ê 逐家有幫贊。

製作：[皓梅](https://howmaywang.com)／顧問：[建中](https://kiantiong.com)

## 練習模式

- **漢字**：共漢字崁起來，看台羅佮解說
- **台羅**：共台羅崁起來，看漢字佮解說
- **解說**：共解說崁起來，看漢字佮台羅

## 回答模式

- **字卡**：揤「看答案」家己評估敢有熟手
- **拍字**：用齒盤輸入答案
